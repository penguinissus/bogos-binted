# Bogos Binted?
This is a game made for Milkyway @ Hackclub!!!  
Current criteria:  
- camera: 2d isometric  
- gameplay: story-based  
- setting: futuristic  

This game will be made with ren'py and the goal of the game will be about getting the photos printed.
